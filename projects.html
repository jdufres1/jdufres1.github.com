This page shows off some of the programming projects that I've been working on in my spare time. I'll upload pictures once I've got 'em, and I will probably also upload the code to github soon as well.

Current projects:

Backyard Wars: A 2D artillery game that features characters who have turned the items usually associated with their careers and hobbies into weapons with which to fight each other... All for fun, of course.

In the picture above: A florist who launches seeds that grow into flowers which attack at the end of every turn, an archer, and a history teacher/Civil War-enthusiast with a cannon. The graphics could use work, but I'm a little bit more concerned with coding at the moment!


My other projects, which aren't as far along, include a RISK-like game that involves insects fighting each other, and an HTML parser that will make it quicker to write blog posts on this website. I could just get an actual blog, but where's the fun in that?!

All of these programs are written in Java.
